============
Installation
============

Requirements
------------

* **Python**: 3.7, 3.8, 3.9, 3.10, 3.11
* **Django**: 2.2, 3.1, 3.2, 4.0
* **Django-filter**: 2.3+

Installation
------------

Install using pip:

.. code-block:: python

    pip install django-property-filter

Then add 'django_property_filter' to your INSTALLED_APPS.

.. code-block:: python

    INSTALLED_APPS = [
        ...
        'django_property_filter',
    ]

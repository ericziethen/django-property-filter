============
Installation
============

Requirements
------------

* **Python**: 3.7, 3.8, 3.9, 3.10, 3.11, 3.12
* **Django**: 3.2, 4.0, 4.1
* **Django-filter**: 23.5+

Installation
------------

Install using pip:

.. code-block:: python

    pip install django-property-filter

Then add 'django_property_filter' to your INSTALLED_APPS.

.. code-block:: python

    INSTALLED_APPS = [
        ...
        'django_property_filter',
    ]

============
Installation
============

Requirements
------------

* **Python**: 3.10, 3.11, 3.12, 3.13, 3.14
* **Django**: 5.2, 6.0+
* **Django-filter**: 24.1+

Installation
------------

Install using pip:

.. code-block:: python

    pip install django-property-filter

Then add 'django_property_filter' to your INSTALLED_APPS.

.. code-block:: python

    INSTALLED_APPS = [
        ...
        'django_property_filter',
    ]
